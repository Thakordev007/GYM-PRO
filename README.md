## 📌 About Project

**GYM PRO** is a full-stack gym management website developed to provide a complete digital platform for gym members and trainers.

The website helps customers explore gym services, check membership plans, connect with trainers, and get fitness guidance through a mini AI assistant.

The project is built using **HTML, CSS, JavaScript, Python Django, and Django Admin Panel**.

---

# 🚀 Features

## 👤 User Features
- User Registration and Login System
- Secure Authentication
- View Gym Membership Plans
- Explore Gym Trainers
- Get Fitness Information
- AI Assistant for Customer Support
- Diet Plan Suggestions
- Fitness Guidance and Recommendations

---

## 🏋️ Trainer Management
- Trainer Details Display
- Trainer Information Management
- Admin can Add/Edit/Delete Trainer Data

---

## 💳 Membership System
- Display Different Membership Plans
- Show Plan Details
- Help Users Select Suitable Gym Packages

---

## 🤖 Mini AI Assistant
- Provides basic customer support
- Suggests diet plans
- Gives workout and fitness information
- Helps users with common fitness queries

---

## 🔐 Django Admin Panel
Admin can manage:

- Users
- Trainers
- Membership Plans
- AI Content/Data
- Website Information
